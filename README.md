# OriginSight — 原生家庭深度回溯

**🔗 访问地址: [https://woodyhouse.github.io/originsight/](https://woodyhouse.github.io/originsight/)**

> **“看见自己，是改变的开始。”**

OriginSight 是一款基于深度心理学逻辑（依恋理论、萨提亚模式）打造的原生家庭性格测评与自我探索工具。它通过回溯童年关键场景，帮助用户觉察潜意识中的生存策略，并提供定制化的疗愈建议。

## ✨ 项目核心

- **🧠 深度测评系统**：基于博主“奇幻日”的扎心逻辑，覆盖从童年创伤、角色错位到成年病理性投射的全维度题库。
- **📊 艺术级雷达图**：动态生成 5 维度性格成因分析，视觉精美且具洞察力。
- **🌊 沉浸式音效引擎**：利用 Web Audio API 纯代码实时合成白噪音，无需外部文件，营造深邃的探索氛围。
- **🎨 氛围感换肤**：页面色调随测评结果实时变换（安全型、焦虑型、回避型、恐惧型）。
- **📸 结果分享海报**：自动生成极简设计的分析海报，方便保存与分享。

## 🛠️ 技术栈

- **前端**: HTML5, Tailwind CSS, JavaScript (ES6+)
- **音频**: Web Audio API (实时合成引擎)
- **图形**: Canvas API (雷达图绘制)
- **存储**: LocalStorage (本地持久化记忆)
